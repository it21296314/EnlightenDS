Talent Prediction API

Use this API to predict talent categories based on user interaction data.

Endpoints:

  • /predict - Make a single prediction
  • /batch_predict - Make predictions for multiple inputs
  • /health - Check API health status
  • /feature_info - Get information about required features

Example Usage:

fetch('/predict', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
    },
    body: JSON.stringify({
        'quiz_time': 600,
        'quiz_score': 4,
        'piano_time': 450,
        'piano_key_count': 200,
        'drawing_time': 900,
        'drawing_similarity': 75,
        'gender': 'F',
        'age': 8
    }),
})
.then(response => response.json())
.then(data => console.log(data));
